Software I wrote

WindowClean North-East

I wrote windowcleannortheast.co.uk. After 5 years of steady growth and pressures related to Covid-19 a website became a priority. I included a registration form (built using Node.js) and other neat features like a services map and before/after images.

DiaryNotes20

After using OneNote I realised that it would be fun to write and use my own notes app for desktop, and later use online. I was able to create a simple but effective app which supports 'Note' objects but also 'Books'. The desktop app uses .json to store notes however I plan to transition to SQL for the WebApp.

PlanetKiller2020

PlanetKiller2020 is an in-browser turn-based strategy game, where you play as the Rebels and aim to capture the 3 secret plans hidden across the galaxy by planning and defeating Empire armies. Each turn the Death Star will destroy one of your planets. Was fun to write a relatively complex game, that I and friends enjoy playing.

Data Analysis

This code reads a .csv file containing data about customer purchases across a 3 year period including average rating and average product value. The aim is to select 100 favourable customers (using all purchase info) who will be sent a voucher.

Creating a Trading Strategy

This code is a simulated stock market where the user can load stock data and portfolios. The automatic trading strategy 'tradeStrategy2' has been extensively tested using real FTSE100 data and will always lead to an (albeit small) profit.
